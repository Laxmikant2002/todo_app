# Django Todo App

This is a simple Todo application built with Django.

## Features

- Add new tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed

## Requirements

- Python 3.x
- Django 5.0.4
- asgiref 3.8.1
- sqlparse 0.5.0
- tzdata 2024.1
